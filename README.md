# Java-Transactions-Processor
A CSV processing tool built using java's Stream API, this project has 0 AI code, it's a simple project in its prototype stage.  
It will get better and more complex with time, for now, it what it is, you might laugh at how simple it is for now, basic OOP  
and functional programming, but it'll get nicer with time dw sweetheart.

## How to Build

### 1. Clone the repository

```bash
git clone https://github.com/suryaanshs007/Java-Transactions-Processor.git
```

### 2. Navigate to the source directory

```bash
cd Java-Transactions-Processor/src
```

### 3. Compile the driver file

Compile only the driver (`Main`) file. The Java compiler will automatically compile all dependent classes.

```bash
javac CsvDataManagerMain.java
```

### 4. Run the program

```bash
java CsvDataManagerMain
```
